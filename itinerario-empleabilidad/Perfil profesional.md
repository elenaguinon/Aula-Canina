## 2. Presentación profesional del proyecto

### Nombre del proyecto

Aula Canina

### Descripción general

Este proyecto consiste en el desarrollo y documentación de un portal web corporativo relacionado con una empresa ficticia. El objetivo principal es aplicar de forma conjunta los conocimientos adquiridos en los módulos de 1º de Desarrollo de Aplicaciones Web.

El proyecto no se limita únicamente a crear una web, sino que también incluye el diseño de una base de datos, una aplicación en Java conectada, documentación técnica, organización del repositorio y una presentación profesional del trabajo realizado.

### Problema que resuelve

El proyecto simula una necesidad real de una empresa: disponer de una presencia digital organizada y de una estructura interna para gestionar su información.

La parte web permite presentar la empresa, sus servicios y su información principal. La parte de programación y base de datos permite representar cómo se gestionan los datos internos del negocio, como clientes, productos, servicios, reservas o cualquier otra entidad relacionada con la temática elegida.

### Público objetivo

El proyecto está pensado para una empresa ficticia que necesita mostrar sus servicios de forma clara y profesional. También está orientado a usuarios que quieran consultar información sobre la empresa, contactar con ella o conocer mejor su actividad.


### Relación con el perfil profesional

Este proyecto representa mi primer acercamiento a un trabajo más parecido al entorno profesional. Me ha permitido entender que un desarrollador web no solo programa, sino que también analiza necesidades, diseña soluciones, organiza archivos, documenta decisiones y presenta su trabajo de forma clara.

Por eso, este proyecto forma parte de mi portfolio inicial como estudiante de Desarrollo de Aplicaciones Web.
