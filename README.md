# Aula Canina


## 1. Descripción del proyecto

**Aula Canina** es un proyecto que simula el desarrollo de una solución digital completa para una empresa del sector del cuidado y la educación canina.

El proyecto integra varias partes que representan un caso real:

- Una **web corporativa** desarrollada con HTML y CSS.
- Una **base de datos relacional** que modela el funcionamiento interno del negocio.
- Una **aplicación de gestión en Java** conectada mediante JDBC.
- Documentación técnica del proyecto.

La web servirá para presentar la escuela canina, mostrar sus servicios y facilitar información a posibles clientes. Por otro lado, la parte de gestión servirá para organizar clientes, perros, servicios y reservas de una forma más ordenada.

---

## 2. Descripcion de la empresa

La empresa ficticia es una **escuela canina**.

Esta escuela ofrecerá servicios relacionados con perros, como por ejemplo:

- Adiestramiento básico.
- Educación para cachorros.
- Psicología canina.
- Clases individuales.
- Clases grupales.
- Deportes caninos.

---

## 3. Objetivo del proyecto

El objetivo principal de **Aula Canina** es desarrollar un portal web y un sistema de gestión que permita:

- Mostrar información clara sobre los servicios de la escuela canina.
- Dar una imagen profesional de la empresa.
- Gestionar clientes y sus mascotas.
- Organizar servicios y reservas.
- Representar una estructura de datos realista del negocio.
- Practicar la conexión entre una aplicación Java y una base de datos MySQL mediante JDBC.


---

## 4. Problema que resuelve

Este proyecto busca solucionar problemas habituales en pequeños negocios como son:

- Gestión manual y desorganizada de clientes.
- Dificultad para controlar reservas y servicios.
- Falta de presencia online.
- Información dispersa o poco accesible.
- Poca claridad para que los clientes conozcan los servicios disponibles.

**Aula Canina** propone una solución digital centralizada que mejora la organización interna del negocio y también la comunicación con los clientes.

---

## 5. A quien va dirgido

El proyecto está dirigido principalmente a:

- Personas que buscan servicios para sus mascotas.
- Clientes de escuelas caninas.
- Dueños de perros interesados en adiestramiento o cuidado profesional.
- Personas que necesitan ayuda con la educación o comportamiento de su perro.

---
## 6. Tecnologías utilizadas

Las tecnologías previstas para el proyecto son:

- **HTML** — estructura del sitio web.
- **CSS** — diseño y maquetación.
- **Java** — lógica de la aplicación de gestión.
- **MySQL** — base de datos relacional.
- **JDBC** — conexión entre Java y la base de datos.
- **Git y GitHub** — control de versiones y organización del proyecto.
- **Markdown** — documentación del repositorio.
