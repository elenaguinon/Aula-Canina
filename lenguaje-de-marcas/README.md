# Lenguajes de Marcas

En esta carpeta se encuentra la parte web del proyecto Aula Canina.

La web representa la parte visible de la empresa. Está desarrollada con HTML y CSS, utilizando una estructura sencilla, clara y organizada.

## Contenido

- Página principal.
- Secciones informativas sobre la empresa.
- Información de servicios.
- Apartado de contacto.
- Estilos CSS personalizados.

## Tecnologías utilizadas

- HTML5
- CSS

## Cómo visualizar la web

Para ver la web, abrir el archivo `index.html` en el navegador.

## Aspectos a tener en cuenta

La web está pensada como portal corporativo básico. Su objetivo es presentar la empresa Aula Canina de forma clara y profesional.
