# Entornos de Desarrollo

En esta carpeta se recoge la parte relacionada con la organización del proyecto y el uso de GitHub.

## Objetivo

El objetivo de este módulo es demostrar una forma de trabajo organizada y parecida a la de un entorno profesional.

## Aspectos trabajados

- Creación del repositorio en GitHub.
- Organización de carpetas.
- Uso de commits.
- Documentación mediante archivos README.
- Limpieza y orden del proyecto.

## Aspectos a tener en cuenta

Cada módulo del proyecto tiene su propia carpeta y una breve explicación para facilitar la revisión y comprensión del trabajo realizado.S
