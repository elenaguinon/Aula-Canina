# Programación

En esta carpeta se encuentra la aplicación Java del proyecto Aula Canina.

La aplicación permite gestionar información relacionada con la empresa.

## Funcionalidades principales

- Inicio de sesión.
- Gestión de datos del negocio.
- Consultas a la base de datos.
- Operaciones básicas de consulta, modificación y eliminación de datos.

## Acceso a la aplicación

Para acceder a la aplicación se utilizan las siguientes credenciales:

- Usuario: `admin`
- Contraseña: `admin123`

## Tecnologías utilizadas

- Java
- JDBC
- SQL

## Aspectos a tener en cuenta

La aplicación está conectada con la base de datos del proyecto.  
Antes de ejecutarla, es necesario tener creada la base de datos y configurada correctamente la conexión JDBC.
